Campus Navigation App
This is a Campus Navigation Application built with Next.js, React, and Tailwind CSS. It provides features for navigating a campus, including maps, search functionalities, and information about departments, facilities, and doctors. The application also includes academic tools and a chatbot.
Features
Campus Map & Navigation: Interactive map with routing capabilities.
Search Functionality: Easily find departments, buildings, facilities, and doctors.
Academic Tools: Credit tracker, workload estimator, graduation checker, course explorer, and curriculum viewer.
Chatbot: AI-powered chatbot for campus-related queries.
Doctor & Facility Information: Detailed views for doctors and facilities, including booking and comparison features.
Getting Started
To run this project locally, follow these steps:
Clone the repository:
Bash
git clone <repository-url>
cd campus-navigation-app
Install dependencies:
Bash
pnpm install
Run the development server:
Bash
pnpm dev
Open http://localhost:3000 with your browser to see the result.
